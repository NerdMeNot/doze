## Three promises

<CardGrid>
  <Card title="Signed, always" icon="approve-check">
    Every module index and archive carries an ed25519 signature; publisher keys
    pin on first use. Tampered or unsigned → rejected, not warned.
  </Card>
  <Card title="Locked, automatically" icon="seti:lock">
    You declare only the engine version. doze selects the newest compatible
    module release and freezes everything in `doze.lock` — commit it, and CI
    runs byte-identical software.
  </Card>
  <Card title="Updated on your say-so" icon="up-caret">
    Pins never drift. `doze modules upgrade --check` reports, `upgrade` moves
    them — and any error that needs a newer module names that exact command.
  </Card>
</CardGrid>
